Credix Database Setup - Instrucciones

Abrir la terminal y conectarse a PostgreSQL:
psql -U <TU_USUARIO> -d postgres
Importante: Reemplaza <TU_USUARIO> por el nombre de usuario de tu máquina.
Ejemplo: psql -U mateovazquezw -d postgres
